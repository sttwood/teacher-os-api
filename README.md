# Teacher OS

Teacher OS คือ Web Application SaaS สำหรับครูราชการไทย  
Phase 1 เริ่มจากระบบ **Plan Builder** สำหรับสร้างแผนการสอนแบบ block/widget-based

## Project Goals

- ช่วยครูสร้างแผนการสอนได้เร็วขึ้น
- ลดการพิมพ์เอกสารซ้ำ ๆ
- รองรับการบันทึก draft และจัดการข้อมูลแผน
- รองรับการ export เป็น DOCX / PDF
- วางโครงสร้างเพื่อรองรับ Marketplace ใน Phase 2
